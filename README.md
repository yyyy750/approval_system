## 🚀 快速开始

### 环境要求

- **JDK** 17+
- **Node.js** 18+
- **pnpm** 8+
- **MySQL** 8.0
- **Maven** 3.9.x

### 1. 克隆项目

```bash
git clone <repository-url>
cd approval_system
```

### 2. 数据库初始化

```bash
# 创建数据库
mysql -u root -p -e "CREATE DATABASE approval_system DEFAULT CHARACTER SET utf8mb4;"

# 导入表结构和初始数据
mysql -u root -p approval_system < script/init_database.sql
```

### 3. 后端启动

```bash
cd backend

# 修改配置文件 (数据库连接信息)
# 编辑 src/main/resources/application.yml

# 运行项目
mvn spring-boot:run
```

后端服务将在 `http://localhost:8080` 启动。

### 4. 前端启动

```bash
cd frontend

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

前端应用将在 `http://localhost:3000` 启动。

## ✨ 核心功能

### 用户角色

| 角色 | 权限说明 |
|------|---------|
| 普通用户 | 发起审批、查看自己的审批记录 |
| 审批人 | 审批待办事项、查看审批历史 |
| 管理员 | 系统配置、用户管理、审批流程配置 |
| 超级管理员 | 所有权限 |

### 审批流程

```
发起申请 → 待审批 → 审批中 → 审批完成
                  ↓          ↓
              已拒绝 ← ─ ─ ─ ┘
                  ↓
              已撤回
```

### 主要页面

- **登录/注册** - 用户认证
- **仪表盘** - 数据概览和快捷操作
- **审批列表** - 查看和管理审批申请
- **发起审批** - 创建新的审批申请
- **审批详情** - 查看审批流程和操作
- **个人资料** - 管理个人信息
- **消息通知** - 查看系统通知
- **成员管理** - 管理员管理用户 (仅管理员)
- **流程配置** - 配置审批流程模板 (仅管理员)
